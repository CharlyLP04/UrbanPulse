# Evidencias de Pruebas Manuales - Imágenes Animadas

## Descripción

Esta carpeta contiene grabaciones/GIFs de las animaciones capturadas durante las pruebas manuales.

## Formatos aceptados

- GIF
- WebM
- MP4
- Captura de pantalla con movimiento

## Grabaciones a crear

### Flujo 1: Apertura de formulario
1. Ir a página principal
2. Click en "Crear Reporte Ahora"
3. Capturar transición suave al formulario

### Flujo 2: Envío de reporte
1. Llenar formulario de prueba
2. Enviar reporte
3. Capturar animación de carga → éxito

### Flujo 3: Estados de carga/error/éxito
1. Capturar cada estado con su animación correspondiente

### Flujo 4: ReportCard
1. Navegar a /public/explore
2. Capturar animaciones de entrada de tarjetas
3. Capturar efecto hover

## Formato de nombres

```
[fecha]_[flujo]_[descripcion].gif
Ejemplo: 2026-03-03_flujo1_apertura-formulario.gif
```

## Herramientas recomendadas

- Windows: OBS Studio, ShareX
- Navegador: Chrome DevTools > Recording
- Online: ezgif.com (para convertir a GIF)
