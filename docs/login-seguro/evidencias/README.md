# Evidencias - Login Seguro

## Descripción

Esta carpeta contiene las evidencias visuales y funcionales de las pruebas realizadas al formulario de login.

## Formatos aceptados

- Video: MP4, WebM, GIF
- Imágenes: PNG, JPG
- Capturas de pantalla

## Contenido esperado

### Video de demostración

- [ ] Video corto (30-60 segundos) demostrando:
  - [ ] Navegación completa por teclado (Tab, Shift+Tab)
  - [ ] Envío de formulario con Enter
  - [ ] Mensajes de error seguros
  - [ ] Redirección tras login exitoso

### Capturas de pantalla

- [ ] Formulario de login con campos vacíos
- [ ] Error de validación de email
- [ ] Error de credenciales inválidas (mensaje genérico)
- [ ] Login exitoso (dashboard)
- [ ] Error 403 al intentar acceder a /admin como USER
- [ ] Acceso correcto a /admin como ADMIN

## Herramientas para grabar

- Windows: OBS Studio, ShareX, Xbox Game Bar (Win+G)
- Navegador: Chrome DevTools > Recording
- Convertidor a GIF: ezgif.com, cloudconvert.com

## Nombre de archivos

```
[fecha]_[tipo]_[descripcion].ext
Ejemplos:
2026-03-17_video_login-teclado.mp4
2026-03-17_captura_error-credenciales.png
2026-03-17_video_rol-admin.mp4
```
